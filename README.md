# 📝 To-Do List App (PWA + APK)

A modern, responsive **To-Do List Application** built using **HTML, CSS, and JavaScript** with advanced features like **Daily Goals, Progress Tracking, Dark Mode, and PWA support**.

🚀 This project is designed to work across **Mobile, Tablet, and Desktop**, and can also be installed as an app or downloaded as an APK.

## ✨ Features

* ✅ Add, Edit, Delete Tasks
* 🎯 Daily Goal Setting
* 📊 Progress Bar (Task Completion Tracking)
* 📅 Automatic Daily Reset
* 🌙 Dark Mode Toggle
* ☰ Hamburger Menu Navigation
* 📥 APK Download Option
* 📱 Fully Responsive Design
* 💾 Local Storage (Data Persistence)
* ⚡ Fast & Lightweight

---

## 📱 Progressive Web App (PWA)

* Install directly from browser
* Works offline using Service Worker
* App-like experience on mobile devices

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Storage:** Browser LocalStorage
* **Deployment:** Vercel
* **APK Hosting:** GitHub Releases / MediaFire
